features

Everything Finder won't tell you

A clean SwiftUI popover on top of libimobiledevice's diagnostic domains — battery internals, storage breakdown, hardware and cellular, in one glance.

🔋

Battery diagnostics

Level, charging state, health %, cycle count, voltage, current, charger wattage/type, design/nominal/full-charge capacity, time remaining.

📦

Storage breakdown

Total & used capacity with a progress bar, plus data and system partition sizes.

📱

Hardware & system

Model, hardware model, CPU architecture, iOS version + build (Beta badge), serial number, activation state, time zone, screen resolution, iCloud backup status.

📡

Cellular, masked by default

Carrier, SIM status, IMEI/IMEI2/ICCID/IMSI, phone number — masked on screen, one click on the eye icon to reveal.

🤝

Trust-flow handling

A dedicated UI state while waiting for "Trust This Computer" — also handles denied trust and a locked device.

🧭

Guided setup

Missing libimobiledevice? The app tells you and gives you the exact brew install command.

🔒

Everything stays on your Mac

iPhoneStatus only talks to your own iPhone over USB via libimobiledevice — no network requests, no analytics, no backend. Sensitive cellular identifiers are masked by default; you choose when to reveal them.

get started

Up and running in a minute

Notarized by Apple — it launches by double-click, no configuration needed.

Download

Grab the latest signed .dmg from GitHub Releases and open it.

Install

Drag iPhoneStatus.app into /Applications.

Plug in your iPhone

Connect over USB and accept "Trust This Computer" — that's it.
